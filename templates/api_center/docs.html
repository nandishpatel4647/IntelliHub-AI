Authentication

IntelliHub uses JWT (JSON Web Tokens) for API authentication. First obtain a token pair, then include the access token in all requests.

Step 1: Get Token

POST {{ base_url }}token/
Content-Type: application/json

{
    "username": "your_username",
    "password": "your_password"
}

Response:

{
    "access": "eyJ0eXAiOiJKV1Qi...",
    "refresh": "eyJ0eXAiOiJKV1Qi..."
}

Step 2: Use in requests

Authorization: Bearer <access_token>
